# The OSI Model


## Physical - Layer 1
Data cables, Cat6 cables, wires etc.
## Data Link - Layer 2
Switching, MAC addresses
## Network - Layer 3
IP addresses, Routing
## Transport - Layer 4
TCP /  UDP
## Session - Layer 5
Session management
## Presentation - Layer 6
WMV, JPEG, MOV (media)
## Application - Layer 7
HTTP, SMTP, etc. the applications we utilise
